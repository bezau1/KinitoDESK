# How to setup KinitiDESK?
Just run the exe from the KinitoDESKPET folder.
# How to setup the CMD command?
1. Make a folder in the C: drive called "KinitoCMD" and put the contents of the KinitoCMD.zip file in the folder
2. Make a desktop Shortcut with the icon being the cmd app icon you can find the image in "%windir%\system32\cmd.exe" when typed to the "Change icon" menu in the properties tab and name it "Command Prompt and set the destination to "C:\KinitoCMD\CMDOPENER.bat"
3. Press win + I to open setting then go to Settings/System/For developers and change Terminal to Windows Console Host
